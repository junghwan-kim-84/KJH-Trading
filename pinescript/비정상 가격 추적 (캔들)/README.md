# MA + cRSI + Vol/Div + ATR Zone (Simple Alerts)

트레이딩뷰에서 사용할 수 있는 Pine Script 지표 설명서입니다.

대상 스크립트:
- [`main.pine`](./main.pine)

## 개요

이 지표는 차트 위에서 아래 내용을 한 번에 보도록 만든 오버레이 스크립트입니다.

- 이동평균선(MA)
- cRSI 기반 과매수 / 과매도 캔들 색상
- 비정상 매수 / 매도 거래량
- RSI 다이버전스
- ATR 기준 배경 구간

지금 구조에서 중요한 점은 `ATR 배경 구간`이 하나의 기준만 쓰는 것이 아니라, 아래 두 가지 기준을 따로 계산한 뒤 합쳐서 보여준다는 점입니다.

- 다이버전스 기준
- 과매수 / 과매도 캔들색 기준

## 트레이딩뷰 적용 방법

1. 트레이딩뷰에서 `Pine Editor`를 엽니다.
2. [`main.pine`](./main.pine) 파일 전체를 복사합니다.
3. Pine Editor에 붙여넣습니다.
4. `차트에 추가`를 누릅니다.
5. 필요하면 저장합니다.

## 기본 정보

- Pine Script 버전: `@version=6`
- 표시 위치: `overlay=true`
- 지표명: `MA + cRSI + Vol/Div + ATR Zone (Simple Alerts)`

## 주요 기능

### 1. 이동평균선(MA)

- `이동평균 기간`
- `평균 기준 가격`
- `종가가 이동평균선 위일 때 색상`
- `종가가 이동평균선 아래일 때 색상`

종가가 MA 위에 있으면 상승 쪽, 아래에 있으면 하락 쪽으로 보기 쉽게 색이 바뀝니다.

### 2. cRSI 과매수 / 과매도 캔들 색상

- `cRSI Dominant Cycle Length`
- `cRSI Vibration`
- `cRSI Leveling (%)`
- `절대 과매수 값`
- `절대 과매도 값`
- `과매수/과매도 시 캔들 색상 변경`
- `과매수 캔들 색상`
- `과매도 캔들 색상`

cRSI가 과열이면 캔들이 과매수 색으로, 너무 눌리면 과매도 색으로 바뀝니다.

### 3. 비정상 거래량 + 스마트 거래량 필터

- `스마트 비정상 거래량 사용`
- `매수/매도 거래량 평균 기준 봉 수`
- `비정상 거래량 배수 기준`
- `거래량 시그널에 MA 필터 적용`
- `거래량 시그널에 cRSI 필터 적용`

이 부분은 단순 거래량 크기만 보는 것이 아니라:

- 매수/매도 성격 분리
- 캔들 몸통/꼬리 구조
- MA 위치
- cRSI 상태

를 함께 써서 `강한 매수 압력봉`, `강한 매도 압력봉`을 거릅니다.

### 4. RSI 다이버전스

- `RSI 다이버전스 표시`
- `RSI 기간`
- `Pivot Right`
- `Pivot Left`
- `최대 탐색 범위`
- `최소 탐색 범위`

지원하는 신호:

- 일반 강세 다이버전스
- 히든 강세 다이버전스
- 일반 약세 다이버전스
- 히든 약세 다이버전스

### 5. ATR 배경 구간

현재 코드 기준으로 ATR 배경 구간은 두 갈래로 계산됩니다.

#### 다이버전스 기준 배경 구간

다이버전스가 나온 뒤 가격이 ATR 기준으로 더 과하게 움직였지만 RSI는 덜 움직인 경우를 감시합니다.

관련 설정:

- `배경 구간 표시`
- `다이버전스 기준 배경 사용`
- `기준 신호 이후 추적 봉 수`
- `ATR 기간`
- `ATR 과도 이동 배수`
- `RSI 추가 이동 제한값`

#### 과매수 / 과매도 캔들 기준 배경 구간

과매수 또는 과매도 캔들색이 나온 뒤에도 같은 방식으로 ATR 과도 이탈 구간을 따로 계산합니다.

관련 설정:

- `과매수/과매도 캔들색 변경 기준 배경 구간 사용`
- `과매수/과매도 캔들 기준 배경 사용`
- `기준 신호 이후 추적 봉 수`
- `ATR 기간`
- `ATR 과도 이동 배수`
- `RSI 추가 이동 제한값`

#### 공통 필터

두 배경 구간 모두 아래 필터를 함께 사용할 수 있습니다.

- `배경 구간에 cRSI 필터 적용`
- `배경 구간에 거래량 필터 적용`

#### 강도별 배경 표현

배경은 한 가지 색만 나오는 것이 아니라 강도별로 나뉩니다.

- 약한 신호: 캔들 기준만 충족
- 중간 신호: 다이버전스 기준만 충족
- 강한 신호: 두 조건을 동시에 충족

그래서 투명도 설정도 각각 따로 있습니다.

- `강세 약한 신호 배경 투명도`
- `강세 중간 신호 배경 투명도`
- `강세 강한 신호 배경 투명도`
- `약세 약한 신호 배경 투명도`
- `약세 중간 신호 배경 투명도`
- `약세 강한 신호 배경 투명도`

## 추천 사용 흐름

1. MA 방향부터 봅니다.
2. cRSI 과매수 / 과매도 캔들이 나왔는지 봅니다.
3. RSI 다이버전스가 붙는지 봅니다.
4. ATR 배경이 약한 신호인지, 중간 신호인지, 강한 신호인지 확인합니다.
5. 거래량까지 같은 방향이면 더 강하게 봅니다.

쉽게 해석하면:

- `상승 추세 + 과매도 캔들 + 강세 다이버전스 + 강세 배경` = 롱 후보
- `하락 추세 + 과매수 캔들 + 약세 다이버전스 + 약세 배경` = 숏 후보

## 알람 설정 방법

이 스크립트에는 `alertcondition(...)`이 포함되어 있어서 트레이딩뷰 알람으로 바로 사용할 수 있습니다.

### 제공되는 알람 종류

- `초기 매수 신호 - 과매도 진입`
- `초기 매도 신호 - 과매수 진입`
- `반복 매수 상태 - 과매도 지속`
- `반복 매도 상태 - 과매수 지속`
- `초기 매수 신호 - 일반 강세 다이버전스`
- `초기 매수 신호 - 히든 강세 다이버전스`
- `초기 매도 신호 - 일반 약세 다이버전스`
- `초기 매도 신호 - 히든 약세 다이버전스`
- `초기 매수 신호 - 통합`
- `초기 매도 신호 - 통합`
- `배경 강세 구간 - ON`
- `배경 약세 구간 - ON`
- `배경 강세 구간 - 강한 신호`
- `배경 약세 구간 - 강한 신호`

## 추가 표시 옵션

필요하면 아래 보조 표시도 켤 수 있습니다.

- `cRSI / 밴드 플롯 표시`
- `매수/매도 거래량 평균선 표시`

## 주의사항

- 자동 매매 전략이 아니라 보조 해석용 지표입니다.
- 다이버전스는 피벗 확정 후 표시되므로 실시간 체감은 늦을 수 있습니다.
- 배경 구간은 많을수록 좋은 것이 아니라, 추세와 거래량이 같은 방향일 때만 우선순위를 높이는 게 맞습니다.
- 종목과 타임프레임에 따라 `ATR 과도 이동 배수`, `RSI 추가 이동 제한값`, `비정상 거래량 배수 기준` 조정이 필요합니다.
