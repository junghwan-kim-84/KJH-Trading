# 비정상 가격 추적 (캔들)

트레이딩뷰용 Pine Script 오버레이 지표 설명서입니다.

대상 스크립트:
- [`abnormal-price-tracker-candles.pine`](./abnormal-price-tracker-candles.pine)

## 개요

이 지표는 차트 위에서 아래 요소를 함께 보여줍니다.

- 이동평균선(MA)
- cRSI 기반 과매수 / 과매도 캔들 색상
- 비정상 매수 / 매도 거래량
- RSI 다이버전스
- ATR 기준 배경 구간
- `L / S` 진입 라벨

`ATR 배경 구간`은 아래 두 기준을 따로 계산한 뒤 합쳐서 보여줍니다.

- 다이버전스 기준
- 과매수 / 과매도 캔들색 기준

## 사용 지표

이 스크립트는 아래 지표를 차트 위에 합쳐서 보여줍니다.

- `Moving Average`
- `cRSI`
- `Buy / Sell Volume`
- `RSI Divergence`
- `ATR` 기반 배경 구간
- `거래량 압력 추적`, `비정상 가격 추적 (보조)`를 합친 `L / S` 진입 신호

## 트레이딩뷰 적용 방법

1. 트레이딩뷰에서 `Pine Editor`를 엽니다.
2. [`abnormal-price-tracker-candles.pine`](./abnormal-price-tracker-candles.pine) 파일 전체를 복사합니다.
3. Pine Editor에 붙여넣습니다.
4. `차트에 추가`를 누릅니다.
5. 필요하면 저장합니다.

## 예시 화면

위 화면에서 핵심으로 보는 요소는 아래와 같습니다.

- 연한 녹색 / 연한 빨간색 9선 MA: 현재 가격이 이동평균선 위인지 아래인지
- 하늘색 / 빨간 캔들: cRSI 기준 과매도 / 과매수
- 위아래 삼각형: RSI 다이버전스
- 배경색: ATR 기준 감시 구간
  강세는 초록, 강한 강세는 파랑, 약세는 주황, 강한 약세는 빨강입니다.

## 이미지에서 어떻게 보는지

- MA가 연한 녹색 쪽이면 상승 우세, 연한 빨간색 쪽이면 하락 우세로 먼저 봅니다.
- 하늘색 과매도 캔들은 `눌림이 깊어진 자리`, 빨간 과매수 캔들은 `반등이 과열된 자리`로 해석합니다.
- 위아래 다이버전스 삼각형은 `힘이 꺾이거나 되돌릴 수 있는 힌트`입니다.
- ATR 배경은 단순 과매수/과매도보다 한 단계 더 강한 감시 구간으로 봅니다.

핵심은 `방향 + 과열/과매도 + 다이버전스 + ATR 배경`이 함께 나오는지 보는 것입니다.
추가로 `L / S` 진입 라벨은 캔들, 거래량, 보조 지표에서 변곡 후보가 잡힌 뒤 실제 반전 확인이 붙는 자리를 한 번 더 추려줍니다.
변곡 후보 유효시간과 진입 확인 시간을 분리하고, 거래량 압력 비율, 보조 지표 gap 축소, 추격 진입 방지, 같은 방향 재발신 제한까지 함께 사용합니다.

## 기본 정보

- Pine Script 버전: `@version=6`
- 표시 위치: `overlay=true`
- 지표명: `abnormal-price-tracker-candles`

## 주요 기능

### 1. 이동평균선(MA)

- `이동평균 기간`
- `평균 기준 가격`
- `종가가 이동평균선 위일 때 색상`
- `종가가 이동평균선 아래일 때 색상`
- `이동평균선 두께`

기본값은 `9선`, 두께 `1`이며 종가가 MA 위에 있으면 연한 녹색, 아래에 있으면 연한 빨간색으로 표시됩니다.

### 2. cRSI 과매수 / 과매도 캔들 색상

- `cRSI Dominant Cycle Length`
- `cRSI Vibration`
- `cRSI Leveling (%)`
- `절대 과매수 값`
- `절대 과매도 값`
- `과매수/과매도 시 캔들 색상 변경`
- `과매수 캔들 색상`
- `과매도 캔들 색상`

cRSI가 과열이면 캔들이 과매수 색으로, 너무 눌리면 과매도 색으로 바뀝니다.

### 3. 비정상 거래량 + 스마트 거래량 필터

- `스마트 비정상 거래량 사용`
- `매수/매도 거래량 평균 기준 봉 수`
- `비정상 거래량 배수 기준`
- `거래량 시그널에 MA 필터 적용`
- `거래량 시그널에 cRSI 필터 적용`

이 기능은 단순 거래량 크기만 보는 것이 아니라 아래 조건을 함께 봅니다.
- 매수/매도 성격 분리
- 캔들 몸통/꼬리 구조
- MA 위치
- cRSI 상태

이 조합으로 `강한 매수 압력봉`, `강한 매도 압력봉`을 거릅니다.

### 4. RSI 다이버전스

- `RSI 다이버전스 표시`
- `RSI 기간`
- `Pivot Right`
- `Pivot Left`
- `최대 탐색 범위`
- `최소 탐색 범위`

지원하는 신호:

- 일반 강세 다이버전스
- 일반 약세 다이버전스

### 5. ATR 배경 구간

ATR 배경 구간은 두 갈래로 계산됩니다.

#### 다이버전스 기준 배경 구간

다이버전스 이후 가격이 ATR 기준으로 더 과하게 움직였지만 RSI는 덜 움직인 구간을 감시합니다.

관련 설정:

- `배경 구간 표시`
- `다이버전스 기준 배경 사용`
- `기준 신호 이후 추적 봉 수`
- `ATR 기간`
- `ATR 과도 이동 배수`
- `강한 배경 판정 ATR 배수`
- `RSI 추가 이동 제한값`

#### 과매수 / 과매도 캔들 기준 배경 구간

과매수 또는 과매도 캔들 발생 뒤에도 같은 방식으로 ATR 과도 이탈 구간을 계산합니다.

관련 설정:

- `과매수/과매도 캔들 기준 배경 사용`
- `기준 신호 이후 추적 봉 수`
- `ATR 기간`
- `ATR 과도 이동 배수`
- `강한 배경 판정 ATR 배수`
- `RSI 추가 이동 제한값`

#### 공통 필터

두 배경 구간 모두 아래 필터를 함께 사용할 수 있습니다.

- `배경 구간에 cRSI 필터 적용`
- `배경 구간에 거래량 필터 적용`

#### 강도별 배경 표현

배경은 네 가지 색으로 구분합니다.

- 강세 배경: 초록
- 강한 강세 배경: 파랑
- 약세 배경: 주황
- 강한 약세 배경: 빨강

강한 배경은 아래 둘 중 하나면 성립합니다.

- 다이버전스 기준 배경과 캔들 기준 배경이 동시에 성립
- 둘 중 하나만 성립해도 가격 이동이 `강한 배경 판정 ATR 배수` 이상으로 커진 경우

투명도는 색상별로 나누지 않고 아래 한 값으로 공통 적용합니다.

- `배경 투명도`

### 6. L / S 진입 라벨

진입 라벨은 아래 3개 축을 함께 점검합니다.

- 캔들 지표의 과매수/과매도, 다이버전스, ATR 배경
- `거래량 압력 추적`의 정렬 + 비정상 매수/매도 압력
- `비정상 가격 추적 (보조)`의 Reversal Risk 확인

기본값은 최근 `변곡 후보 유효시간` 안에서 3개 축 중 2개 이상이 같은 방향이면 먼저 변곡 후보로 보고,
그 뒤 `진입 확인 봉 수` 안에서 cRSI 회복이나 `close > high[1]` / `close < low[1]` 돌파 확인이 붙을 때 `L` 또는 `S` 라벨을 표시합니다.
거래량 축은 `진입 거래량 압력 비율`을 통과해야 하고, 보조 축은 `cRSI-MFI gap`이 줄어드는 구간만 인정합니다.
또한 가격이 MA나 최근 변곡 후보 저점/고점에서 너무 멀어지면 추격 진입으로 보고 라벨을 막고,
한 번 `L` 또는 `S`가 발생하면 반대 후보가 나오기 전까지 같은 방향 라벨을 `같은 방향 재발신 제한 봉 수` 동안 막습니다.
`추세 추적`은 계산에 넣지 않고, 사용자가 따로 방향 판단에 참고하는 기준으로 둡니다.

관련 설정:

- `L/S 진입 라벨 표시`
- `변곡 후보 유효시간`
- `진입 확인 봉 수`
- `변곡점 축 최소 충족 수`
- `진입 거래량 압력 비율`
- `L 라벨 색상`
- `S 라벨 색상`
- `L/S 라벨 ATR 간격`
- `추격 진입 제한 MA 거리(ATR)`
- `추격 진입 제한 변곡점 이탈(ATR)`
- `같은 방향 재발신 제한 봉 수`

## 추천 사용 흐름

1. MA 방향부터 봅니다.
2. cRSI 과매수 / 과매도 캔들이 나왔는지 봅니다.
3. RSI 다이버전스가 붙는지 봅니다.
4. ATR 배경이 일반 배경인지, 강한 배경인지 확인합니다.
5. 거래량까지 같은 방향이면 더 강하게 봅니다.

- `상승 추세 + 과매도 캔들 + 강세 다이버전스 + 강세 배경` = 롱 후보
- `하락 추세 + 과매수 캔들 + 약세 다이버전스 + 약세 배경` = 숏 후보

## 거래 예시

### 롱 예시

- MA가 상승 쪽으로 유지되고 가격도 MA 위 또는 MA 근처 지지를 받습니다.
- 눌림 구간에서 하늘색 과매도 캔들이 나옵니다.
- 같은 자리 또는 직후에 강세 다이버전스와 강세 ATR 배경이 붙습니다.
- 이때 보조 지표가 `Bullish Reversal Risk` 라벨, 거래량 지표가 비정상 매수 거래량을 같이 보여주면 롱 진입 후보로 봅니다.

### 숏 예시

- MA가 하락 쪽으로 유지되고 가격이 MA 아래에서 반등하다 막힙니다.
- 반등 구간에서 빨간 과매수 캔들이 나옵니다.
- 같은 자리 또는 직후에 약세 다이버전스와 약세 ATR 배경이 붙습니다.
- 이때 보조 지표가 `Bearish Reversal Risk` 라벨, 거래량 지표가 비정상 매도 거래량을 같이 보여주면 숏 진입 후보로 봅니다.

## 알람 설정 방법

이 스크립트에는 `alertcondition(...)`이 포함되어 있어 트레이딩뷰 알람으로 바로 사용할 수 있습니다.

### 제공되는 알람 종류

- `매수 후보 신호 - 과매도 시작`
- `매도 후보 신호 - 과매수 시작`
- `매수 후보 유지 - 과매도 계속`
- `매도 후보 유지 - 과매수 계속`
- `매수 후보 신호 - 하락 힘 약해짐`
- `매도 후보 신호 - 상승 힘 약해짐`
- `매수 감시 구간 진입`
- `매도 감시 구간 진입`
- `매수 감시 구간 강화`
- `매도 감시 구간 강화`
- `롱 진입 신호 - L`
- `숏 진입 신호 - S`

## 추가 표시 옵션

필요하면 아래 보조 표시를 켤 수 있습니다.

- `cRSI / 밴드 플롯 표시`
- `매수/매도 거래량 평균선 표시`

## 주의사항

- 자동 매매 전략이 아니라 보조 해석용 지표입니다.
- 다이버전스는 피벗 확정 후 표시되므로 실시간 체감은 늦을 수 있습니다.
- 배경 구간은 많을수록 좋은 것이 아니며, 추세와 거래량이 같은 방향일 때 우선순위를 높이는 것이 좋습니다.
- 종목과 타임프레임에 따라 `ATR 과도 이동 배수`, `RSI 추가 이동 제한값`, `비정상 거래량 배수 기준` 조정이 필요합니다.
