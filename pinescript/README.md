# PineScript 트레이딩 가이드

이 폴더는 여러 지표를 같이 보고 `롱`, `숏`, `관망`을 판단하기 위한 정리입니다.

핵심만 먼저 보면:

- `상승 추세 + 눌림 + 매수 거래량`이면 롱 후보
- `하락 추세 + 반등 실패 + 매도 거래량`이면 숏 후보
- 신호가 섞이면 관망

초보자는 `애매하면 안 들어가는 것`이 가장 중요합니다.

![전체 예시 화면](./img.png)

## 보는 순서

1. [`추세 추적`](./추세%20추적/README.md)  
   지금 시장의 큰 방향을 봅니다.  
   `HH/HL`이면 상승 쪽, `LH/LL`이면 하락 쪽으로 봅니다.

2. [`비정상 가격 추적 (보조)`](./비정상%20가격%20추적%20(보조)/README.md)  
   지금 자리가 과열인지, 과매도인지 봅니다.  
   바닥 반등 후보나 고점 꺾임 후보를 찾을 때 봅니다.

3. [`비정상 가격 추적 (캔들)`](./비정상%20가격%20추적%20(캔들)/README.md)  
   실제 차트 위에서 진입 후보 자리를 확인합니다.  
   다이버전스, 과매수/과매도, ATR 배경 구간을 봅니다.

4. [`비정상 거래량 추적`](./비정상%20거래량%20추적/README.md)  
   마지막으로 거래량에 힘이 붙었는지 확인합니다.  
   롱이면 매수 거래량, 숏이면 매도 거래량을 봅니다.

## 아주 쉬운 해석

- `롱`: 위로 가는 흐름에서 잠깐 눌렸다가 다시 오를 때
- `숏`: 아래로 가는 흐름에서 잠깐 반등했다가 다시 밀릴 때
- `관망`: 방향, 자리, 거래량 중 하나라도 애매할 때

## 마지막 원칙

- 모르면 진입하지 않기
- 손절 자리를 먼저 정하기
- 신호 하나만 보고 진입하지 않기
- 애매하면 관망하기

돈이 걸려 있다면, `좋은 진입을 더 찾는 것`보다 `나쁜 진입을 피하는 것`이 먼저입니다.
