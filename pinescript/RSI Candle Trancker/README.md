# MA + cRSI + Vol/Div + ATR Zone

트레이딩뷰에서 사용할 수 있는 Pine Script 지표 설명서입니다.

대상 스크립트:
- [`main.pine`](/Users/kjh/Workspace/KJH-Trading/pinescript/RSI%20Candle%20Trancker/main.pine)

## 개요

이 지표는 하나의 오버레이 스크립트 안에 아래 기능을 함께 제공합니다.

- 이동평균선(MA)과 종가 위치에 따른 색상 변화
- cRSI 기반 과매수/과매도 판별 및 캔들 색상 강조
- 비정상 매수/매도 거래량 감지
- RSI 다이버전스 표시
- 다이버전스 이후 ATR 기준 과도 이탈 구간 배경 표시

차트 위에서 추세, 모멘텀, 거래량, 다이버전스, 되돌림 구간을 한 번에 확인하려는 목적에 맞춰 구성된 스크립트입니다.

## 트레이딩뷰 적용 방법

1. 트레이딩뷰에서 `Pine Editor`를 엽니다.
2. [`main.pine`](/Users/kjh/Workspace/KJH-Trading/pinescript/RSI%20Candle%20Trancker/main.pine) 파일의 전체 내용을 복사합니다.
3. Pine Editor에 붙여넣습니다.
4. `Add to chart` 또는 `차트에 추가`를 클릭합니다.
5. 필요하면 `Save`로 개인 스크립트로 저장합니다.

## 기본 정보

- Pine Script 버전: `@version=6`
- 표시 위치: `overlay=true`
- 지표명: `MA + cRSI + Vol/Div + ATR Zone`

## 주요 기능

### 1. 이동평균선(MA)

- `maLength`: 이동평균 기간
- `src`: 평균 계산 기준 가격
- 종가가 MA 위에 있으면 `colorAbove`, 아래에 있으면 `colorBelow` 색으로 표시됩니다.

### 2. cRSI 과매수 / 과매도

- `domCycle`, `vibration`, `leveling`: cRSI 계산 민감도 관련 설정
- `absoluteOB`, `absoluteOS`: 절대 과매수/과매도 기준값
- `paintBars`: 조건 충족 시 캔들 색상 변경

조건이 충족되면:
- 과매수 구간은 `overboughtColor`
- 과매도 구간은 `oversoldColor`

### 3. 비정상 거래량 감지

- `enableSmartVolume`: 스마트 거래량 시그널 사용 여부
- `abnormalVolLookback`: 평균 거래량 계산 기준 봉 수
- `abnormalVolMultiplier`: 평균 대비 비정상 거래량 판단 배수
- `useMAFilterForVolume`: MA 필터 적용 여부
- `useCRSIFilterForVolume`: cRSI 필터 적용 여부

캔들의 몸통과 꼬리 구조를 반영해서 매수 압력봉 / 매도 압력봉을 구분하고, 평균 대비 비정상 거래량 발생 여부를 판단합니다.

### 4. RSI 다이버전스

- `showRSIDivergence`: 전체 다이버전스 표시 여부
- `rsiDivLength`: RSI 기간
- `lbR`, `lbL`: 피벗 탐지 좌우 범위
- `rangeUpper`, `rangeLower`: 비교 가능한 다이버전스 탐색 범위

지원하는 다이버전스:
- 일반 강세 다이버전스
- 히든 강세 다이버전스
- 일반 약세 다이버전스
- 히든 약세 다이버전스

강세는 차트 하단 삼각형, 약세는 차트 상단 삼각형으로 표시됩니다.

### 5. ATR Setup Zone

다이버전스가 나온 뒤 가격이 ATR 기준으로 과도하게 더 움직였지만 RSI는 상대적으로 덜 움직인 구간을 배경색으로 강조합니다.

주요 설정:
- `showSetupZone`
- `setupLookbackBars`
- `atrLength`
- `atrMultiplier`
- `rsiMoveLimit`
- `useCRSISetupFilter`
- `useVolumeSetupFilter`

표시 방식:
- 강세 후보 구간: `setupBullBgColor`
- 약세 후보 구간: `setupBearBgColor`

## 추천 사용 흐름

1. MA로 현재 추세 방향을 먼저 확인합니다.
2. cRSI 과매수/과매도 상태를 봅니다.
3. 비정상 거래량이 함께 발생하는지 확인합니다.
4. RSI 다이버전스가 있는지 확인합니다.
5. ATR Setup Zone이 표시되면 되돌림 또는 반전 후보 구간으로 추가 검토합니다.

이 지표는 확정 진입 신호보다는 후보 구간 탐색 용도로 보는 편이 더 적절합니다.

## 알람 설정 방법

이 스크립트는 트레이딩뷰의 `Create Alert` 기능에서 바로 사용할 수 있도록 `alertcondition(...)`이 포함되어 있습니다.

### 트레이딩뷰에서 알람 만드는 순서

1. 차트에 이 지표를 추가합니다.
2. 상단의 `Alert` 또는 알람 아이콘을 클릭합니다.
3. `Condition` 항목에서 이 스크립트 이름인 `MA + cRSI + Vol/Div + ATR Zone`을 선택합니다.
4. 오른쪽 조건 목록에서 원하는 알람 항목을 선택합니다.
5. 알람 빈도, 만료 시간, 알림 방식(앱, 팝업, 웹훅 등)을 설정합니다.
6. `Create`를 눌러 저장합니다.

### 제공되는 알람 종류

- `cRSI 과매수 진입`: cRSI가 과매수 기준값을 새로 상향 돌파할 때
- `cRSI 과매도 진입`: cRSI가 과매도 기준값을 새로 하향 이탈할 때
- `cRSI 과매수 반복`: 과매수 상태가 유지되는 동안 봉 마감 기준으로 반복 감지
- `cRSI 과매도 반복`: 과매도 상태가 유지되는 동안 봉 마감 기준으로 반복 감지
- `RSI 일반 강세 다이버전스`: 일반 강세 다이버전스 발생 시
- `RSI 히든 강세 다이버전스`: 히든 강세 다이버전스 발생 시
- `RSI 일반 약세 다이버전스`: 일반 약세 다이버전스 발생 시
- `RSI 히든 약세 다이버전스`: 히든 약세 다이버전스 발생 시
- `통합 매수 계열 알림`: 과매도 진입 또는 강세 다이버전스 발생 시
- `통합 매도 계열 알림`: 과매수 진입 또는 약세 다이버전스 발생 시
- `통합 매수 계열 반복 알림`: 과매도 상태 지속 또는 강세 다이버전스 발생 시
- `통합 매도 계열 반복 알림`: 과매수 상태 지속 또는 약세 다이버전스 발생 시
- `기본 강세 ATR 배경 구간`: 강세 다이버전스 이후 ATR 기준 감시 구간 진입 시
- `기본 약세 ATR 배경 구간`: 약세 다이버전스 이후 ATR 기준 감시 구간 진입 시

### 알람 해석 팁

- `진입` 알람은 조건이 새로 발생하는 시점 확인에 적합합니다.
- `반복` 알람은 특정 상태가 계속 유지되는 동안 추적 감시에 적합합니다.
- `통합 매수/매도 계열` 알람은 여러 신호를 하나로 묶어서 받고 싶을 때 유용합니다.
- `ATR 배경 구간` 알람은 다이버전스 이후 되돌림 또는 반전 후보 구간 감시에 적합합니다.

### 추천 설정

- 빠른 알림이 필요하면 짧은 타임프레임에서 `Once Per Bar Close` 기준으로 사용
- 노이즈를 줄이려면 `통합 매수 계열 알림`, `통합 매도 계열 알림` 위주로 사용
- 되돌림 감시가 목적이면 `기본 강세 ATR 배경 구간`, `기본 약세 ATR 배경 구간`을 별도로 생성
- 반복 알람이 너무 많으면 스크립트 설정에서 `과매수/과매도 반복 알림 사용` 옵션을 끌 수 있음

## 주의사항

- 자동 매매 전략이 아니라 시각화 중심의 보조지표입니다.
- 다이버전스는 피벗 확정 이후 표시되므로 실시간으로는 늦게 확정될 수 있습니다.
- 종목, 타임프레임에 따라 `atrMultiplier`, `rsiMoveLimit`, `abnormalVolMultiplier` 조정이 필요합니다.
- 거래량이 왜곡되는 시장이나 저유동성 종목에서는 신호 품질이 낮아질 수 있습니다.
