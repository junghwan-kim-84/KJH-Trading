# 거래량 압력 추적

트레이딩뷰에서 사용할 수 있는 Pine Script 지표 설명서입니다.

대상 스크립트:
- [`volume-pressure-tracker.pine`](./volume-pressure-tracker.pine)

## 개요

이 지표는 거래량을 `Buy Volume`과 `Sell Volume`으로 나눠서, 현재 시장의 `거래량 압력`이 어느 쪽에 더 실리는지 보여줍니다.

여기에 `비정상 거래량`과 `추세 정렬 배경`까지 함께 표시해서, 단순 거래량 크기보다 `실제 매수/매도 압력`을 더 해석하기 쉽게 만든 지표입니다.

## 사용 지표

이 스크립트는 아래 지표를 조합해서 거래량 압력과 방향 정렬을 봅니다.

- `Buy Volume / Sell Volume`
- `Abnormal Volume`
- `Dynamic Swing Anchored VWAP`
- `Ichimoku Base Line`
- `Simple Moving Average`

## 트레이딩뷰 적용 방법

1. 트레이딩뷰에서 `Pine Editor`를 엽니다.
2. [`volume-pressure-tracker.pine`](./volume-pressure-tracker.pine) 파일 전체를 복사합니다.
3. Pine Editor에 붙여넣습니다.
4. `차트에 추가`를 누릅니다.
5. 필요하면 저장합니다.

위 이미지에서 볼 수 있는 핵심은 아래 3가지입니다.

- 흰색/회색 막대: 일반 매수/매도 거래량
- 연두색/빨간색 막대: 비정상 거래량이 붙은 매수/매도 압력 구간
- 초록색/자주색 배경: 추세 정렬이 맞는 구간

쉽게 해석하면:

- `막대 색`은 거래량 압력의 성격
- `배경 색`은 시장 방향의 정렬 상태

## 이미지에서 어떻게 보는지

이미지에서 이 지표는 `마지막 거래량 압력 확인`에 가깝습니다.

- 초록 배경이면 상승 정렬, 자주 배경이면 하락 정렬입니다.
- 연두색 비정상 거래량은 `평소보다 강한 매수 유입`으로 봅니다.
- 빨간 비정상 거래량은 `평소보다 강한 매도 압력`으로 봅니다.

핵심은 거래량 막대 하나보다 `배경 방향과 같은 쪽의 압력이 비정상적으로 커지는지`입니다.

- 초록 배경 + 연두 막대 = 롱 쪽 힘 확인
- 자주 배경 + 빨간 막대 = 숏 쪽 힘 확인
- 배경과 막대가 반대 = 추격보다 관망 우선

## 기본 정보

- Pine Script 버전: `@version=6`
- 표시 위치: 별도 패널
- 포맷: `format.volume`
- 지표명: `Buy/Sell Volume + Abnormal Volume + Alignment Background`

## 주요 기능

### 1. Buy / Sell Volume 분리

캔들 구조를 바탕으로 거래량을 매수 압력과 매도 압력으로 나눠 보여줍니다.

- 종가가 봉 아래보다 위에서 마감할수록 `Buy Volume`이 커집니다.
- 종가가 봉 위쪽보다 아래에서 마감할수록 `Sell Volume` 성격이 강해집니다.

### 2. 비정상 거래량 표시

현재 거래량이 최근 평균보다 충분히 크면 비정상 거래량으로 표시합니다.

주요 설정:

- `비교 캔들 수`
- `비정상 배수`

쉽게 말해, `평소보다 유난히 큰 거래량 압력`이 붙은 봉을 바로 찾는 기능입니다.

### 3. Alignment Background

이번 버전에서 추가된 핵심 기능입니다.

내부적으로 아래 기준을 계산해서 배경색을 보여줍니다.

- 스윙 기준 적응형 VWAP
- Ichimoku Base Line
- 이동평균선(MA)

정렬 조건은 아래와 같습니다.

- 강세 정렬: `MA > Base Line > VWAP`
- 약세 정렬: `MA < Base Line < VWAP`

차트 해석은 아주 단순하게 하면 됩니다.

- 초록 배경: 상승 쪽 정렬이 맞는 구간
- 자주 배경: 하락 쪽 정렬이 맞는 구간
- 배경 진하기: 정렬 간격이 넓을수록 더 진하게 표시되며 `약 / 중 / 강` 3단계 투명도로 구분됩니다.

강도는 `MA-Base Line` 간격과 `Base Line-VWAP` 간격 중 더 약한 쪽을 기준으로 계산합니다.

- 약한 정렬: 아주 연한 배경
- 중간 정렬: 눈에 띄는 중간 배경
- 강한 정렬: 확실히 진한 배경

## 추천 사용 흐름

1. 먼저 배경색으로 시장 정렬이 위인지 아래인지 봅니다.
2. 그다음 거래량 막대 색으로 매수/매도 압력을 봅니다.
3. 정렬과 거래량 압력 방향이 같을 때만 신호를 강하게 봅니다.

예시:

- `초록 배경 + 강한 매수 압력` = 롱 쪽 참고
- `자주 배경 + 강한 매도 압력` = 숏 쪽 참고
- `배경과 거래량 압력이 반대` = 애매하므로 약하게 보기

## 거래 예시

### 롱 예시

- 추세 지표가 `HH/HL` 구조이고 초록 정렬이 유지됩니다.
- 캔들 지표에서 눌림 이후 강세 신호가 나옵니다.
- 이 거래량 지표에서 초록 배경 안에 연두색 비정상 매수 압력 막대가 터집니다.
- 그러면 `상승 정렬 안에서 실제 매수 참여가 붙었다`고 해석하고 롱 쪽 확률을 높게 봅니다.

### 숏 예시

- 추세 지표가 `LH/LL` 구조이고 자주 정렬이 유지됩니다.
- 캔들 지표에서 반등 이후 약세 신호가 나옵니다.
- 이 거래량 지표에서 자주 배경 안에 빨간 비정상 매도 압력 막대가 터집니다.
- 그러면 `하락 정렬 안에서 실제 매도 참여가 붙었다`고 해석하고 숏 쪽 확률을 높게 봅니다.

즉, 이 지표는 방향을 처음 정하는 도구가 아니라 `이미 잡아둔 시나리오에 실제 힘이 붙는지 확인하는 도구`로 쓰면 가장 좋습니다.

## 주의사항

- 이 지표는 자동 진입 신호가 아니라 보조 해석용입니다.
- 거래량 색만 보고 진입하지 말고, 배경 정렬과 가격 구조를 같이 봐야 합니다.
- `비정상 배수`를 너무 낮추면 색이 너무 자주 바뀔 수 있습니다.
- 횡보장에서는 배경색 신호가 자주 바뀌어 해석이 어려워질 수 있습니다.
